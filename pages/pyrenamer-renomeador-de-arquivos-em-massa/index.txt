Um renomeador de arquivos em lote é uma ferramenta que não pode faltar em seu desktop, seja linux, windows ou macOS, no linux pode mos fazer isso via console, para quem é familiarizado com o shell, isso não é um problema, mas como essa habilidade não é para todos devemos então disponibilizar ferramentas gráficas para tal, nesse momento eu apresento o **pyRenamer**.

Com pyRenamer você pode alterar o nome de vários arquivos ao mesmo tempo facilmente.

- Usar padrões para renomear arquivos.
- Usar search & replace para renomear arquivos.
- Usar substituições comuns.
- Renomear manualmente os arquivos selecionados.
- Renomear imagens usando seus metadados.
- Renomear músicas usando seus metadados.

**padrões**: Padroniza a alteração de nomes, incluindo variáveis internas que você pode usar nos padrões, como adicionar números, nome do diretório, data, ano, mês e assim por diante, bem como substituições de números aleatórios;

**substituições**: Fornece algumas substituições comuns como espaços, traços , capitalização letras maiúsculas, minúsculas, primeira letra maiúscula ou primeira letra de cada palavra em maiúsculas etc.

**inserir / excluir**: Esta opção, permite inserir ou excluir caracteres de nomes de arquivos em uma determinada posição;
**imagens e música**: Esta opção renomea imagens e arquivos de áudio com base em seus metadados.

**Arch Linux / Manjaro / Antergos**

```
yay -S  pyrenamer
```

**Debian / Ubuntu / Linux Mint**

```
sudo apt-get install pyrenamer
```
